# vue-rem-layout-start


## intro
A vue-cli3-based mobile rem layout initial project


### vue-cli3 Version

@vue/cli 3.0.0-rc.8


## Pre-rendering tool

- *html*:  
pug

- *js*:  
typescript

- *css*:  
less

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
